# CHRONOS | Welterbe & Weltwunder Expedition

Eine interaktive, minimalistische Weltkarte zur Erkundung der bedeutendsten Kulturgüter, Naturerbe-Stätten und Weltwunder der Menschheit. Gefiltert nach Epochen, sortiert nach Geschichte und verpackt in ein modernes, barrierefreies User Interface.

## ✨ Features

- **Modernist Minimal Design:** Ein von modernen Betriebssystemen inspiriertes, klares sans-serif UI (Inter / Source Sans 3) mit einem ultra-cleanen Hellmodus.
- **Weltwunder-Klassifikation:** Stätten, die sowohl den UNESCO-Status als auch den Titel eines Weltwunders tragen, werden priorisiert als Weltwunder deklariert. Ein tiefgründiger Analysetext beleuchtet den dualen Status.
- **Erweiterte Datenbank:** Detaillierte historische Analysen, Hintergrundberichte und architektonische Besonderheiten für alle Stätten – inklusive Neuzugängen wie der Akropolis, Angkor Wat und dem Great Barrier Reef.
- **Expeditions-Prüfung:** Integrierte Gamification-Elemente mit Wissensfragen zu jeder Stätte, um Forscher-Punkte (XP) und neue Ränge freizuschalten.
- **Virtuelle Kamera-Touren:** Automatisierte Kamera-Führungen, die die topografischen Schlüsselpunkte der Stätten sequenziell auf der Karte anfliegen.
